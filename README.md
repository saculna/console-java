  ```shell
  faça um programa que leia 5 nomes
  depois mostre os 5 nomes na tela.
  ```  

   ## Exercicio 2

  ```shell
  Danilo, professor de programação está precisando de um software para cadastrar
  os seus aluns e dar as notas respectivas para cada aluno e depois calcular
  a media para saber se estes alunos estão aprovados, de recuperação ou
  reprovados.
  Faça um programa que cadastre:
  1 - alunos 
  2 - notas dos alunos
  3 - mostre se o aluno está aprovado, em recuperação ou reprovado

  Regra de aprovação:
  Notas abaixo de 5 = reprovado
  Notas entre 5 e 6 = recuperação
  Notas 7 para cima = aprovado

  Mostre uma opção para vermos o relatório dos alunos cadastrados.
  ```
